# audit-workflow

Используй для полного health check Python-проекта.

## Разделы аудита
1. Архитектура и boundaries
2. Безопасность и secrets handling
3. Качество кода и тестов
4. Dependency hygiene
5. Operational readiness

## Результат
Сформируй consolidated report с health score 0-10, списком рисков, quick wins и long-term improvements.
