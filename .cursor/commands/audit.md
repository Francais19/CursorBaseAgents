# /audit

Используй для полного health check Python-проекта.

## Workflow
1. Senior Reviewer — архитектура
2. Security Auditor — безопасность
3. Reviewer — качество кода и тестов
4. Documenter — consolidated report + health score
5. Опционально: Debugger для критичных fixes после подтверждения

## Выход
- score 0-10
- top risks
- quick wins
- strategic improvements
