# /implement

Используй для простой Python-задачи: функция, endpoint, schema, service-method, CLI-command, test helper.

## Workflow
1. Передай задачу Worker.
2. Если изменения нетривиальны — попроси Test Writer добавить/обновить тесты.
3. Передай результат Test Runner для проверки.
4. Передай результат Documenter для краткого отчета, если docs enabled.

## Выход
- код
- тесты
- краткий report: что изменено и как проверено
