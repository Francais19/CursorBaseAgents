# /refactor

Используй для безопасного рефакторинга Python-кода.

## Workflow
1. Senior Reviewer определяет smells и цели.
2. Refactor делает изменения малыми шагами.
3. Test Runner верифицирует поведение.
4. При проблемах Debugger исправляет.
5. Documenter пишет report.

## Условие
До старта должны существовать тесты или должен быть согласован шаг по их добавлению.
